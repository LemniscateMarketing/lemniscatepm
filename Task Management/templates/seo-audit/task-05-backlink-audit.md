## Context

A backlink audit is optional for small/new sites with thin link profiles but
essential for clients experiencing ranking instability or Google penalties.

**Client:** PLACEHOLDER_CLIENT
**Website:** PLACEHOLDER_DOMAIN

This task is marked medium priority because on-page and technical issues
typically have faster ROI than link building. However, a toxic backlink profile
can suppress the entire site — verify early.

> **Note:** If the client is a new site with few backlinks, this task can be
> descoped. Check with the project lead before starting. See TEMPLATE.md.

## Checklist

**Profile Overview**
- [ ] Pull full backlink profile (Ahrefs, Semrush, or Moz)
- [ ] Note: total backlinks, referring domains, domain authority distribution
- [ ] Compare link velocity over last 12 months (growth, flat, or decay?)

**Quality Assessment**
- [ ] Identify top 20 links by authority — are they legit?
- [ ] Flag links from spammy or irrelevant sites (spam score > 30)
- [ ] Check anchor text distribution: over-optimised? natural? branded?
- [ ] Identify toxic or manipulative links (PBNs, link farms, unnatural patterns)

**Lost Links**
- [ ] Check for recently lost high-authority links — can they be reclaimed?
- [ ] Identify broken pages that had inbound links (redirect opportunities)

**Competitor Benchmark**
- [ ] Compare referring domain count to top 3 competitors
- [ ] Identify link sources that link to competitors but not the client

**Disavow (if needed)**
- [ ] Build disavow file if penalty or toxic link pattern found
- [ ] Get sign-off from project lead before submitting to Google

## Acceptance Criteria

- Backlink health score documented with clear Good / Watch / Act classification
- Toxic links list prepared (if any found)
- Disavow file ready for submission (if applicable)
- Competitor link gap: at least 5 potential link acquisition targets identified
